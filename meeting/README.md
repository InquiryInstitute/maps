# Boardroom Map

Official board meeting room for the Inquiry Institute Council of Inquiry.

## Access

The boardroom map is available at:
```
https://grounds.inquiry.institute/_/global/inquiryinstitute.github.io/maps/boardroom
```

Or directly:
```
https://grounds.inquiry.institute/_/global/inquiryinstitute.github.io/maps/boardroom/map.json
```

## Features

- **10 Director Seats**: 5 on each side of the conference table
- **Heretic Seat**: At the head of the table (Diogenes)
- **Parliamentarian Position**: At the bottom, unseated
- **Matrix Chat Integration**: Automatically connects to `#boardroom:inquiry.institute`

## Files

- `map.tmj` - Tiled map file (visual design)
- `map.json` - WorkAdventure map file (WAM format)
- `boardroom-script.js` - Script for seating and Matrix integration
- `boardroom-map-setup.md` - Detailed setup documentation

## Matrix Room

- **Room ID**: `!PXnmXLlzxpsYHbQidS:matrix.inquiry.institute`

## Usage

1. Navigate to the boardroom URL
2. Enter your name and select a Woka
3. Matrix chat will automatically connect
4. Board members will be placed in their designated seats
